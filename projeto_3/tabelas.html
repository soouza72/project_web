O elemento table (tabela)

O elemento table (tabela) reprenseta dados tubulares ou seja, informações que são representados em linhas e colunas

Exemplos

Tabela simples com cabeçalho

Elementos da tabela

  • table - tabela
  • th (table hearder) - cabeçalho da coluna.
  • tr (table row) - linha da tabela.
  • td (table data) - valor/conteúdo da célula.
